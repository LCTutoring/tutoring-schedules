Tutor Schedule Finder 
=================================================

This folder is made to be easy to edit 

FILES IN THIS FOLDER
--------------------
1. index.html
   - Main webpage
   - Change page text, headings, and labels here

2. style.css
   - Change colors, font sizes, spacing, and visual design here

3. data.js
   - Change tutor schedule information here
   - This is the file you edit when tutor names, days, or times change

4. script.js
   - Controls search behavior and results shown on the page
   - Only edit this if you want to change how the page works

MOST COMMON CHANGES
-------------------
If you want to...

- change the page title:
  open index.html

- change the font size:
  open style.css
  look for: body { font-size: ... }

- change orange or other colors:
  open style.css
  look for the :root section near the top

- update a tutor's schedule:
  open data.js

- change the weekday search:
  open data.js or script.js

HOSTING
-------
If you upload this site to GitHub Pages or another web host, the file that opens first is:

    index.html

IMPORTANT
---------
Keep all files together in the same folder:
- index.html
- style.css
- script.js
- data.js

If one file is missing, the page may not work correctly.
